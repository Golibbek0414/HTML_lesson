Massivlar

Massiv tushunchasi. Massivni e'lon qilish va ishga tushirish


Massiv-bir xil turdagi elementlarning qat'iy uzunlikdagi ketma-ketligi.
Massiv quyidagicha e'lon qilinadi:
var a [3]int
fmt.Println(a) // [0 0 0]

Massiv deklaratsiyasi bilan bir vaqtda biz uning elementlarining qiymatlarini o'rnatishimiz mumkin


var a [3]int = [3]int{1, 2, 3}
b := [3]int{1, 2, 3}
c :=[...]int{1, 2, 3}
d :=[3]int{1: 12}

fmt.Println(a) //[1 2 3]
fmt.Println(b) //[1 2 3]
fmt.Println(c) //[1 2 3]
fmt.Println(d) //[0 12 0]

Massivlarni taqqoslash

Massivning barcha elementarini ketma-ket taqqoslashimiz mumkin bo'lganligi sababli,biz massivlarning o'zini solishtirishimiz mumkin:


a := [3]int{1, 2, 3}
b := [3]int{1, 2, 3}
c := [3]int{3, 2, 1}

fmt.Println(a==b) // true
fmt.Println(a==c) // false
!!!! Ammo shuni hisobga olish kerakki, faqat bir xil turdagi massivlar solishtirish mumkin (bir xil uzunlikdagi bir xil turdagi elementlarni o'z ichiga olgan massivlar).
